# Ukiyo Themes (Obsidian → VS Code)

Port of the Obsidian **Ukiyo** theme color schemes to **Visual Studio Code**.

## Included schemes
- Cozy Typewriter (Default)
- Ukiyo
- Golden Coffee
- Golden Book (Dark only)
- Alchemy
- Rose Shadow (Dark only)

## Install (local)
1. Clone this repo
2. Run `npm i -g @vscode/vsce` (optional)
3. Package: `vsce package`
4. Install the generated `.vsix`
